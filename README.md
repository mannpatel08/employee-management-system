# Employee Management System

## Description
Offline React CRUD app using localStorage.

## Features
- Add, Edit, Delete Employees
- Data persistence using localStorage
- Conditional UI rendering
- Event handling and props

## Tech Stack
- React
- Vite
- JavaScript

## Run Project
npm install  
npm run dev
